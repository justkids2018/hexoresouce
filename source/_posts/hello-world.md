### 创建命令

``` bash
$ hexo new "My New Post" //文章
$ hexo new page “测试”   //页面
```

### HEXO 运行命令

``` bash
$ hexo clean  //清理
$ hexo generate  //编译
$ hexo server   //服务器
$ hexo deploy   //发布
```
